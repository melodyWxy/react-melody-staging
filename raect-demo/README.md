## React Easy Start
基于create-react-app搭建的React项目脚手架。
## Installation
```
git clone https://github.com/sunnut/react-easy-start.git
cd react-easy-start
npm install
npm run mock
npm start
```  
## Dependencies
* React 16.7      --hooks
* Redux           --状态管理
* React-Router-V4 --多页面
* Ant Design      --UI
* React Intl      --国际化
* Gulp            --合并国际化词条
* Restify         --Mock服务器
